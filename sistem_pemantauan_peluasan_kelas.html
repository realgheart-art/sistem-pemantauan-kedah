📊 Cara Setup Integrasi Google Sheets

  1. Buat Google Sheet baru di Google Drive
  2. Klik Extensions → Apps Script
  3. Copy dan paste kod di bawah ke dalam Apps Script editor
  4. Klik Deploy → New deployment → Web app
  5. Pilih "Anyone" untuk access, kemudian Deploy
  6. Copy Web App URL yang diberi
  7. Paste URL tersebut ke dalam kotak "Google Sheets URL" di bawah
